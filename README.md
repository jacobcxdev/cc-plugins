# claude-plugins

Jacob Clayden's Claude Code plugin marketplace.

## Installation

```
/plugin marketplace add jacobcxdev/claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [model-matchmaker](https://github.com/jacobcxdev/model-matchmaker) | Effort-aware prompt routing for Claude Code |

## Installing a Plugin

```
/plugin install model-matchmaker@jacobcxdev-plugins
```
